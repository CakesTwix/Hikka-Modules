## Moebooru

| Name                                                         | Description                                           |
| ------------------------------------------------------------ | ----------------------------------------------------- |
| [moebooru](https://gitlab.com/CakesTwix/friendly-userbot-modules/-/tree/master/CustomROMs) [ROOT] | Module for obtaining art from the ImageBoard Moebooru |

## Commands

| **Command**   | Description         |
| ------------- | ------------------- |
| .mlast        | The last posted art |
| **Arguments** |                     |
| -t            | Get tags with art   |

| **Command**   | Description       |
| ------------- | ----------------- |
| .mrandom      | Random posted art |
| **Arguments** |                   |
| -t            | Get tags with art |
